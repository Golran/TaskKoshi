# Численное решение задачи коши 
## Моя задача: 
y`=50*y*(x-0.6)(x-0.85), 0<=x<=1
y(0)=0.1 
## Методы: 
* Явный метод Эйлера
* Метод Рунге-Кутта 4го порядка
* Метод трапеций
## Что стоит менять в проекте
* В классе Algorithms переименуйте мои методы так, чтобы мои методы были переименованы в ваши по всему проекту(в Visual Studio клавиша F2).
* В классе Algorithms реализуйте свои методы так, чтобы всё работало.
* В папке Settings в классе PaletteSettings измените название полей, они отвечают за цвет графиков различных методов.
* Аналогично в  папке Settings в классе GraphicsSettings измените название полей, они отвечают отрисовку графика того или иного метода.
* Переименовываем в последних 2х пунктах так же как и в 1ом.
## exe
Exe файл после компиляции будет лежать в ...DifferentialEquations\DifferentialEquations\bin\Release.
Или он может лежать в ...DifferentialEquations\DifferentialEquations\bin\Release.
Всё зависит от того, в каком режиме вы собирали проект.
